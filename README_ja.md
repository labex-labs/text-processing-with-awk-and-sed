AWK と sed によるテキスト処理を学びます。このモジュールでは、sed の置換の基本、sed のストリーム編集、AWK のカラム抽出、AWK のフィルタリングとロジック、AWK の計算を扱います。実践的なラボと実世界の課題を通じて、これらの必須の Linux スキルを習得します。
